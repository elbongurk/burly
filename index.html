Does typing <?php echo what_was_that_function ?> make your head hurt?
Burly tags are small blocks of code like {%raw%}{{burlyText food}}{%endraw%} that when inserted into HTML files indicate specific areas need to be editable from within the WordPress administrative panel.
Ever want more than just a WordPress editor to give to the client?
There are several different Burly tag types, each one indicates to WordPress what type of content is present, and therefore, what type of input to display in the administrative panel.
Wish you had more control of the design?
These tags are used to replace the content (text, image, link, etc) of your HTML in a way that you have complete control. With Burly, you do not have to compromise your site design to make your content editable.
So how do you get started?
Head over to the Quickstart to find out how to get up and running with Burly. Afterwards, use the Reference documenation to dive a bit deeper. If you think something is broken or could be improved, submit an issue on Github.
